# 🌐 Logigraph – Network System
<br/>
Logigraph is a simple project where I tried to apply graph algorithms to something practical like routing and logistics. The idea was to move beyond theory and actually see how BFS and DFS can be used in real-world scenarios.
<br/>

## 🚀 Features
<br/>
• Models a network using nodes and edges <br/>
• Finds routes using BFS and DFS <br/>
• Demonstrates basic graph traversal <br/>
• Keeps the implementation simple and readable <br/>

## ⚙️ Tech Stack
<br/>
JavaScript (Node.js) <br/>
